What is CSS Grid layout?

CSS Grid can be used to achieve many different layouts. It excels at dividing a page into major regions, or defining the relationship in terms of size, position, and layer, between parts of a control built from HTML primitives.

Unlike tables, grid layout doesn't have content structure, therefore enabling a wide variety of layouts not possible in tables.

For example, a grid container's child elements could position themselves so they actually overlap and layer, similar to CSS positioned elements.
